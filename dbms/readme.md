# Toy DBMS

## Query syntax

```sql
SELECT column_name_1, column_name_2 FROM table_name

SELECT column_name_1, column_name_2 FROM table_name WHERE !(column_name_1 >= 123)

INSERT INTO table_name VALUES column_name_1, value_1, column_name_2, value_2

DELETE FROM table_name WHERE column_name_1 == 123 || column_name_2 == "asd"
```
